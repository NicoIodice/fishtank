## 2026-06-04 — IA & Shell

| # | Decision | Value |
|---|---|---|
| 17 | Sidebar order (initial) | Services → Network Activity → Mappings → Mock Suggestions → [divider] → Warnings & Errors → Settings |
| 18 | Top bar (always visible) | Logo/brand left · Right side: About (ℹ️), Notification bell (badge), User account icon |
| 19 | Notification bell | Badge = unread count · Click opens dropdown panel · Paginated: 10 at a time, scroll loads next 10 · Mark-as-read decrements badge · Notifications sourced from Warnings & Errors events |
| 20 | Warnings & Errors surface | Dedicated tab for: duplicate GUIDs, malformed mappings/responses, config conflicts, import mismatches, engine crash events, filesystem write failures |

| 23 | Network Activity row detail | Default: Modal (B). User-configurable in Settings: Modal / Right Drawer / Bottom Panel |

| 25 | Final sidebar items | Services → Network Activity → Mappings → [divider] → Warnings & Errors → Settings (Mock Suggestions removed as nav item) |
| 26 | Notification bell scope | Errors + warnings AND info events (Record mode enabled, Mock saved, Resync completed) |
| 27 | "Warnings & Errors" nav item renamed | [ASSUMPTION: "System Events"] — two internal tabs: "Warnings & Errors" + "Info" · single flat list per tab (no sub-type tabs) |
| 28 | Services default view | Card grid; toggle to table; each card: name, description, port badge, upstream URL, Live/Stopped toggle + indicator |
| 29 | Mappings editor | Form + Raw JSON tab toggle; both always available |
| 30 | Screens to design | Services, System Events, Settings, Mappings (all in one interactive mockup) |

## Features / IA (confirmed)
- Services Management
- Network Activity (real-time log, filters, row detail — configurable pattern)
  - inline: Save as Mock action per proxied row
  - inline: Record Mode toggle (auto-promote proxied → mappings)
- Mappings & Responses (service-scoped, pick service first)
- Warnings & Errors (persistent log)
- Notification Center (bell — paginated, mark-as-read)
- Top Bar (About ℹ, Notifications 🔔, User avatar)
- Settings (global config, theme switcher, row detail style preference, users/auth)
- Auth (login, JWT, admin setup)
